# Calculator Project

## Project Overview
This is a **web-based calculator** implemented in JavaScript. It supports basic arithmetic operations, percentages, powers, square roots, and a **history of the last 10 calculations**. The calculator handles **keyboard input** as well as button clicks, making it easy and intuitive to use.

## Features
- Addition, subtraction, multiplication, division
- Exponentiation (`^`) and square root (`√`)
- Percentage calculations
- Handling negative numbers (`±`)
- Supports parentheses for complex expressions
- **History panel** showing the last 10 calculations
- Keyboard input support

---

## Controls

### Buttons
- `AC` — Clear the current expression
- `←` — Delete last character
- `=` — Evaluate the expression
- `±` — Toggle sign of the current number
- `√` — Insert square root
- `%` — Percentage operation
- Number buttons `0–9` and operators `+`, `−`, `×`, `÷`, `^`, `(`, `)`

### Keyboard
| Key        | Action |
|------------|--------|
| `0–9`      | Input number |
| `+ - * /`  | Arithmetic operations |
| `Enter` or `=` | Evaluate expression |
| `Backspace` | Delete last character |
| `c`        | Clear expression |
| `r`        | Insert square root (`√(`) |
| `m`        | Toggle sign of last number |
| `h`        | Toggle history panel visibility |
